登录接口

URL地址 site/login
接收参数 username，password
返回结果 1-成功，0-登录账号或密码有误

退出接口

URL地址 site/logout

验证是否登录

URL地址 site/islogin
返回结果 1:已登录，0：未登录

修改密码

URL地址 site/update
接收参数 password 新密码
返回结果 -1:密码为空，1:修改成功，0：修改失败
